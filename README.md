## ⚙️ Installation

Repository klonen:

```bash
git clone https://github.com/dein-benutzername/gaming-blog.git
```

In das Projektverzeichnis wechseln:

```bash
cd gaming-blog
```

Abhängigkeiten installieren:

```bash
npm install
```

Entwicklungsserver starten:

```bash
npm run dev
```

Die Anwendung ist anschließend unter:

```text
http://localhost:5173
```

erreichbar.
